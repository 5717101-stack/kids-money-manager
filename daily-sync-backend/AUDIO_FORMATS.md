# 🎵 פורמטי אודיו נתמכים

## ✅ פורמט מומלץ: WAV

**WAV תמיד עובד** - ללא תלויות נוספות!

### איך להמיר ל-WAV:

#### ב-Mac (עם QuickTime):
1. פתח את הקובץ ב-QuickTime
2. File → Export As → Audio Only
3. בחר WAV

#### ב-Mac (עם Audacity - חינמי):
1. הורד Audacity: https://www.audacityteam.org/
2. פתח את הקובץ
3. File → Export → Export as WAV

#### אונליין (ללא התקנה):
- https://cloudconvert.com/ - המרה אונליין
- https://convertio.co/ - המרה אונליין

## ⚠️ פורמטים אחרים:

- **MP3** - דורש codecs (ffmpeg/ffprobe)
- **M4A** - דורש codecs
- **FLAC** - דורש codecs
- **OGG** - דורש codecs
